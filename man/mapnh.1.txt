.TH MAPNH 1 LOCAL

.SH NAME

mapnh - Substitution mapping and branch clustering according to substitution process

.SH SYNOPSIS

.B mapnh [options]

.SH AVAILABILITY

All UNIX flavors

.SH DESCRIPTION

mapnh map substitutions on a phylogenetic tree and cluster branches according to the underlying substitution process.

.SH OPTIONS

You should refer to 'info testnh' or to the online manual of testnh for a complete list of available options.

.TP 5

--noninteractive

generates output for redirection in a file.

.TP

param=file

reads a file for loading options
